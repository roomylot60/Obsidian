## Class 생성 (in python)
### Inheritance(상속)

```python
class Class_name:
```

- 일반적인 class 생성에서는 class 명을 선언하면서 생성됨
- 이 때, 상속을 활용하여 타 class; parent class를  동시에 선언함으로써 부모 class의 attributes와 methods를 사용 가능

```python
class Class_name(Parent_class_name):
```

#### Example

```python
# inheritance.py
class Country:
    """Super Class"""

    name = '국가명'
    population = '인구'
    capital = '수도'

    def show(self):
        print('국가 클래스의 메소드입니다.')


class Korea(Country):
    """Sub Class"""

    def __init__(self, name):
        self.name = name

    def show_name(self):
        print('국가 이름은 : ', self.name)
```

```REPL
>>> from inheritance import *
>>> a = Korea('대한민국')
>>> a.show()
국가 클래스의 메소드입니다.
>>> a.show_name()
국가 이름은 :  대한민국
>>> a.capital
'수도'
>>> a.name
'대한민국'
```

- `inheritance.py` 파일으로부터 `Country`, `Korea` class를 불러옴
- `Korea`의 객체 `a`에 대해 `'대한민국'`이라는 parameter를 입력
